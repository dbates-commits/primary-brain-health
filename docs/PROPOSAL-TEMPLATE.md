- Testing
  - We use the following tools to ensure a high-quality end product:
    - Google Lighthouse
      - Lighthouse is an open-source, automated tool to help you improve the quality of web pages.
      - We take an average score after reviewing it 4 times in incognito mode and aim for a minimum of 80 or higher. In most cases we can get the score much higher, Performance in particular being the most intricate to optimize.
    - PageSpeed Insights
      - PageSpeed Insights reports on the user experience of a page on both mobile and desktop devices, and provides suggestions on how that page may be improved.
    - Core Web Vitals
      - Core Web Vitals are a set of metrics used by Google to assess a website's user experience, focusing on speed, interactivity, and visual stability, and are part of Google's "page experience" ranking signals.
  - Calculating Scores
    - We take an average score after reviewing it 4 times in incognito mode and aim for a minimum of 80 or higher. In most cases we can get the numbers much higher with performance typically being the most time consuming to optimize.
    - In our experience, the more third-party scripts we load on the page the harder it can be to achieve scores of 90+ in the Performance category.
  - We'll run these tools throughout development and again once the site has been pushed to production. We'll implement the recommended improvements to enhance site speed and user experience.
  - Browser Compatibility
    - To ensure our web pages deliver a consistent and optimal user experience across various platforms, we will conduct tests on the following browsers:
