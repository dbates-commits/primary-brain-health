Tina CMS is a powerful, Git-backed headless CMS that provides a seamless editing experience for your Next.js applications. In this guide, we'll walk through setting up Tina from scratch.

## Why Choose Tina CMS?

Unlike traditional headless CMS solutions, Tina stores your content directly in your Git repository. This means you get version control, branching, and collaboration features out of the box.

## Installation

Getting started is simple. Just run the following command in your Next.js project:

```bash
npx @tinacms/cli@latest init
```

This will set up the basic configuration and create the necessary files to get you started.

## Next Steps

Once installed, you can start defining your content schema and building your components. The visual editor makes it easy to see changes in real-time.
